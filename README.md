# Budgeting App MVP

This application allows users to create a budget, create a transaction, view all budgets, and view all transactions in order to manage, plan, and forecast the company's budgets. 

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. 

### Installing

A step by step of how to get the development env running

Clone the repository

```
git clone https://github.com/antoniomelendez/budgetapp.git
```

Navigate to the project

```
cd budgetapp
```

Install to the project

```
npm install
```

Start GraphQL server 

```
npm start
```

Server will run on 

```
localhost:4000/graphql
```

Start react app

```
cd client
npm install
```

React app will run on 

```
localhost:3000
```

## Built With

* [React](https://reactjs.org/) - The framework used
* [GraphQL](https://graphql.org/) - The server used 
* [Apollo Client](https://www.apollographql.com/client/) - The client used to communicate with the GraphQL server
